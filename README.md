# -windows

Для чтения создать вручную файл размером несколько гигибайт, желательно больше размера оперативной памяти. 
Сравнить результаты на механическом жёстком диске, USB Flash диске и, по возможности, SSD на высокопроизводительном интерфейсе (PCI-E)
